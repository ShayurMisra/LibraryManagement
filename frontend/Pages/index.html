Overview

Welcome to the Library Management System! This website is designed to help you manage your library efficiently. Below is a brief guide on how to use the different sections of the site:

Book Management

To manage books, click on the Books tab. Here, you can:

  • Add Books: Fill out the form to add new books to the library.
  • View Books: Scroll down to view all books currently in the library.
  • Edit Book Details: Click the Edit button next to a book to update its details. An edit form will open for you to make changes.
  • Delete Books: Click the Delete button to remove a book from the library.

User Management

For user management, go to the Users tab. This section provides similar functionality to the books section:

  • Add Users: Use the form to add new users.
  • View Users: See a list of all users by scrolling down.
  • Edit User Details: Click the Edit button next to a user to update their details.
  • Delete Users: Click the Delete button to remove a user from the system.

Transactions

To manage book transactions, navigate to the Transactions tab:

  • Add Transaction: Record a new transaction for lending or returning books.
  • Update Transactions: Modify existing transactions as needed.
  • Close Transactions: Complete transactions to mark books as returned.

Search

Use the Search tab to find:

  • Books: Search by book title, or ISBN.
  • Users: Look up users by mobile number.

Logout

When you're finished, click the Logout button to exit the system.
